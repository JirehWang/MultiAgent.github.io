# General Light

Use for reversible tasks where speed and cost matter more than deep reasoning.

Avoid this profile for debugging, architecture, compliance, broad code changes, or high-stakes review.

When routing provides `compression_policy`, this profile may run with both settings low or off on tiny tasks, but clarity still wins over forced terseness.
