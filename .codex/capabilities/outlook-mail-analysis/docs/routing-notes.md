# Routing Notes

Route Outlook and mailbox-analysis requests here when they involve one or more of:

- exported Outlook folders
- `.eml` or `.msg` batch analysis
- email reply timing
- unreplied threads
- thread-level summaries
- sender-specific mail filtering
- subject-based mail search over a report
- report-first mailbox workflows

Prefer this capability over general writing or manual file reading when the request is about analyzing a whole mail dataset, not drafting one specific reply.
