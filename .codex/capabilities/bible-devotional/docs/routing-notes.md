# Routing Notes

Route Bible-related requests here when they involve one or more of:

- Bible passage study
- exegesis
- authorial intent
- biblical theology
- redemptive history synthesis
- spiritual formation
- devotional writing
- discussion questions for a passage
- sermon-support interpretation
- Bible maps or FHL-locked slide workflows

Prefer the devotional orchestrator stack over generic planning or writing skills when the request is clearly Scripture-centered.

Default staged order:

1. exegesis
2. biblical theology
3. spiritual formation
4. discussion and synthesis
5. optional slide or map packaging
