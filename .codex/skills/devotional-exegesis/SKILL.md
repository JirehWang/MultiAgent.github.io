# Devotional Exegesis

Work in author-first mode.

## Core rule

Answer this before anything else:

- What is the author saying?
- Why is the author saying it here?

Do not drift into:

- devotional sentiment
- sermon rhetoric
- direct application
- speculative typology

## Minimum analysis grid

Always cover:

1. `Context`
   - historical background
   - literary context
   - geographic context when relevant
   - immediate before/after context
2. `Structure`
   - major section placement
   - internal movement of the passage
3. `Theme`
   - what burden or emphasis dominates the unit
4. `Literary Flow`
   - why the material is arranged in this order
   - what tension, contrast, repetition, or progression is doing the work
5. `Authorial Intent`
   - what question the author is really answering
6. `Keywords`
   - repeated words
   - repeated concepts
   - hinge terms or transitions

## Discipline rules

- Prefer textual logic over generic background dumping.
- If a long section looks boring on the surface, ask why the author gave it so much space.
- Name the controlling question in one sentence.
- If the user gives a large block, identify the most natural unit before explaining it.
- If structure is uncertain, give one primary reading rather than many weak options.

## Output contract

Use these headings in this order:

- `Context`
- `Structure`
- `Theme`
- `Literary Flow`
- `Authorial Intent`
- `Keywords`

Never end this stage with:

- `today we should`
- direct exhortation
- discussion questions

Those belong to later skills.
