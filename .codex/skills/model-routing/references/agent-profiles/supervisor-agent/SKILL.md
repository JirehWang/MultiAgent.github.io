# Supervisor Agent

Use for QA-first oversight rather than primary execution.

Prefer this profile when the main question is whether an agent run was acceptable, efficient, on-plan, and ready to pass a completion gate.

This role is especially appropriate for:

- AWU review and governance
- plan-vs-actual assessment
- trajectory inspection
- retry or escalation decisions
- completion claim validation
